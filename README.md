# Motion Detection System using Arduino

## Description
An IoT-based project that detects motion using a PIR sensor and triggers alerts using a buzzer and LCD display.

## Features
- Real-time motion detection
- Buzzer alert system
- LCD display for motion status

## Components Used
- Arduino
- PIR Sensor
- Buzzer
- LCD Display
